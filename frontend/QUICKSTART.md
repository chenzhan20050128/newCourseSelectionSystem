# 快速启动指南

## 前置要求

1. Node.js (版本 14 或更高)
2. npm 或 yarn
3. 后端服务已启动并运行在 `http://localhost:8080`

## 启动步骤

### 1. 进入前端目录

```bash
cd frontend
```

### 2. 安装依赖

```bash
npm install
```

如果安装速度慢，可以使用国内镜像：

```bash
npm install --registry=https://registry.npmmirror.com
```

### 3. 启动开发服务器

```bash
npm run dev
```

### 4. 访问前端

打开浏览器访问：`http://localhost:3000`

## 测试功能

### 课程查询测试

1. 在"课程查询"标签页
2. 填写任意查询条件（所有字段都是可选的）
3. 例如：输入"学院"为"计算机学院"，"校区"为"本部"
4. 点击"查询"按钮
5. 查看返回的课程列表和节次信息

### 节次查询测试

1. 切换到"节次查询"标签页
2. 选择"周几"（如：周三）
3. 输入"开始节次"（如：5）
4. 输入"结束节次"（如：5）
5. 点击"查询"按钮
6. 查看返回的课程ID列表

## 常见问题

### 1. 无法连接到后端

- 确保后端服务已启动
- 检查后端是否运行在 `http://localhost:8080`
- 查看浏览器控制台的错误信息

### 2. CORS错误

- 后端已配置CORS，如果仍有问题，检查后端CorsConfig配置
- 确保使用Vite代理（已在vite.config.js中配置）

### 3. 端口被占用

- 如果3000端口被占用，Vite会自动使用下一个可用端口
- 或者修改 `vite.config.js` 中的 `server.port` 配置
