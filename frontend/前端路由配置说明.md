# 前端路由配置说明

## 更新内容

已为项目添加了 Vue Router，实现登录页面和主页面的分离。

## 页面路由

- `/` - 自动重定向到登录页
- `/login` - 登录页面
- `/home` - 主页面（需要登录）

## 路由守卫

系统会自动检查登录状态：
- 未登录访问 `/home` 会重定向到 `/login`
- 已登录访问 `/login` 会重定向到 `/home`

## 安装步骤

### 1. 安装依赖

进入前端目录并安装 vue-router：

```bash
cd frontend
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

### 3. 访问应用

浏览器打开 `http://localhost:5173`，会自动显示登录页面。

## 新增文件

1. **router/index.js** - 路由配置文件
2. **views/Home.vue** - 主页面组件（包含原来的课程查询功能）
3. **components/Login.vue** - 登录组件（已存在）

## 修改文件

1. **App.vue** - 简化为路由容器
2. **main.js** - 添加路由支持
3. **package.json** - 添加 vue-router 依赖

## 功能特性

### 登录页面
- 用户名/密码输入
- 用户类型选择（学生/教师）
- 错误提示
- 自动跳转

### 主页面
- 显示用户信息
- 退出登录按钮
- 课程查询
- 节次查询
- 我的课程（仅学生）
- 智能选课（仅学生）

## 用户体验

1. 首次访问 → 显示登录页面
2. 登录成功 → 跳转到主页面
3. 已登录用户直接访问根路径 → 自动跳转到主页面
4. 点击退出登录 → 清除登录信息，返回登录页面
5. 未登录用户尝试访问主页 → 自动跳转到登录页面
